Portfolio logo preview

Right-click the logo and choose "Save image as..." to download the SVG. To upload to LinkedIn Featured you should use a PNG (800×800 or 1200×1200). See the instructions below to convert the SVG to PNG on Windows.

Convert SVG → PNG (recommended)

  1. If you have ImageMagick installed, open PowerShell in this folder and run:
    magick convert .\\images\\portfolio-logo.svg -resize 1200x1200 .\\images\\portfolio-logo.png
  2. If you don't have ImageMagick, use Chrome: open this file in Chrome, right-click the SVG element in DevTools (Elements) and choose "Capture node screenshot" — that saves a PNG at the displayed resolution.
  3. Or use an online converter (CloudConvert, svg2png, etc.) and export 1200×1200 PNG.

When you have the PNG, upload it to LinkedIn Featured → Add media → select the PNG file. Title: My Portfolio. Description: Personal portfolio showcasing projects and skills.
